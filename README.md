# gazebo_sample

## Installation
```
$ cd [your catkin workspace]
$ git clone https://github.com/chibi314/gazebo_sample
$ catkin build
```

## Run
Default (dark)
```
$ roslaunch gazebo_sample bringup.launch world:=1
```

Scene parameters modified (light)
```
$ roslaunch gazebo_sample bringup.launch world:=2
```
